(Armazenar em um objeto, e inserir esse objeto em um array de restaurantes)

Cada objeto precisa de um metodo para calcular a gorjeta. retornar, armazenar o valor calculado
junto do objeto original

Cada objeto tem que calcular o valor total (conta + gorjeta)

Cada objeto tem que ter um metodo toString() que retorna uma string assim ->
"Restaurante do Ze Birola - [Valor: - R$60.00 | gojeta: | total: ]"

metodo no Array para calcular o total gasto

metodo no array para calcular a media de gastos totais

metodo no array que retorna o objeto do reustauranto com maior gasto
